MODULE: Administrator Module
EPIC  Authentication & Access Control
User Story: As an admin, I want to log in securely so that I can access the system.
Tasks:
Design login UI (email + password fields, forgot password link)
Develop login API endpoint (POST /api/auth/login)
Implement JWT-based authentication and session management
Validate credentials against hashed passwords in the database
Handle error messages (invalid credentials, inactive account)
Implement logout functionality and token invalidation
Write test cases for login and logout flows

EPIC  Staff Management
User Story: As an admin, I want to create new staff profiles so that I can onboard team members with appropriate access.
Tasks:
Design Add Staff UI form (Name, Contact, Gender, Address, Salary, EmpID, Role)
Develop Create Staff API (POST /api/staff)
Implement form validation (required fields, format checks)
Hash password before storing in TblStaff
Assign role to staff from TblRole on creation
Write test case: valid data → 201 Created
Write test case: missing fields → 400 Bad Request

User Story: As an admin, I want to update existing staff records so that I can keep their information current.
Tasks:
Design Edit Staff UI with pre-filled form
Develop Update Staff API (PUT /api/staff/{staffId})
Validate updated fields before persisting
Prevent duplicate username/email updates
Write test case: valid update → 200 OK

User Story: As an admin, I want to deactivate staff accounts so that I can revoke system access when needed.
Tasks:
Add Deactivate button in staff list UI
Develop Deactivate Staff API (PATCH /api/staff/{staffId}/deactivate)
Set IsActive = 0 in TblStaff (soft delete)
Prevent deactivated staff from logging in
Write test case: valid deactivation → 200 OK

User Story: As an admin, I want to view and search the staff list so that I can manage all staff members efficiently.
Tasks:
Design Staff List UI with search bar and Edit/Delete buttons
Develop List All Staff API (GET /api/staff)
Develop Get Staff by ID API (GET /api/staff/{staffId})
Implement search/filter by name or role
Write test case: retrieve all staff → 200 OK with list
